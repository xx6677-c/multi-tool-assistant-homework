# 知简笔记 · 功能与快捷键

> 本文档为教学用**虚构**资料，所有功能与快捷键均为示例。

## 常用快捷键

| 操作 | 快捷键 |
| --- | --- |
| 新建笔记 | Ctrl+N |
| 全局搜索 | Ctrl+Shift+F |
| 插入待办项 | Ctrl+T |
| 切换暗色模式 | Ctrl+Shift+D |

笔记**自动保存**，无需手动保存快捷键。

## 编辑功能

- Markdown 实时预览。
- 表格与代码块语法高亮。
- 手写涂鸦（仅 iPad 端支持）。

## 导入与导出

- **导出**格式：Markdown（.md）、PDF、HTML。
- **导入**来源：Evernote（.enex 文件）、Notion 导出包。

## 组织方式

知简笔记用三种方式组织内容：笔记本、标签、双向链接。
