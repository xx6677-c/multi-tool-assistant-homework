# 知简笔记 · 产品概览

> 本文档为教学用**虚构**资料，「知简笔记」并非真实产品，所有信息均为示例。

知简笔记（ZhiJian Notes）是一款面向个人与团队的跨平台笔记应用。当前最新版本为 **v4.2**，于 **2026 年 3 月** 发布。

## 它能做什么

- 用 Markdown 实时编辑笔记，支持表格、代码块高亮。
- 用「笔记本 + 标签 + 双向链接」三种方式组织内容。
- 端到端加密同步，多设备随时访问。
- 专业版起支持 AI 摘要，自动为长笔记生成要点。

## 支持的平台

知简笔记提供 Web 版，以及 Windows、macOS、iOS、Android 客户端，共五个平台。
